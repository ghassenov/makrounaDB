# Scripts

## Development

- `scripts/dev/build.sh` - configure and build with CMake + Ninja
- `scripts/dev/test.sh` - run ctest suite
- `scripts/dev/run.sh [port] [aof_path]` - start makrounaDB CLI server
- `scripts/dev/bench.sh ...` - run benchmark harness

## CI

- `scripts/ci/run_ci.sh` - build and run all tests
